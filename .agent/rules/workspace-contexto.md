# 🏢 Contexto do Workspace: Oficina Baby Steps com IA

Este repositório é destinado à construção e organização da oficina **"Programando com IA: Como Desenvolvedores Trabalham Hoje"**, apresentada na **FADERGS** (#fazfadergs). O objetivo é servir como material de apoio para introduzir calouros ao uso de IA no desenvolvimento de software.

## 🎯 Objetivo do Projeto
Criar um ambiente de aprendizado focado em:
*   Desmistificar a **Inteligência Artificial** para iniciantes.
*   Apresentar o conceito de **Agente de IA** (vs Chatbot).
*   Dar uma **experiência prática** onde os alunos criam algo real com IA.
*   Mostrar como profissionais de tecnologia já trabalham com IA no dia a dia.

## 🛠️ Ferramentas Principais

### 📊 Marp (Markdown Presentation Ecosystem)
Estamos utilizando o **Marp** para criar apresentações de slides diretamente a partir de arquivos Markdown.
*   **Arquivo Principal**: `slides.md` (na raiz ou pastas específicas).
*   **Skill Disponível**: Consulte `.agent/skills/marp-slides-creator/SKILL.md` para ver como criar e estilizar slides.
*   **Uso**: O Marp permite converter markdown simples em slides visuais com temas (Gaia, Default, Uncover) e personalizações CSS.

## 📂 Estrutura Importante
*   **.agent/**: Configurações do agente (Antigravity/Google).
*   **.cursor/rules/**: Configurações equivalentes para o Cursor AI.
*   **slides.md**: Apresentação principal da oficina.
